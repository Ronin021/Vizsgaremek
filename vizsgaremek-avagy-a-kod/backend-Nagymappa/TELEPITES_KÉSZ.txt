NPM CSOMAGOK TELEPÍTVE:
=======================

Fő függőségek (Dependencies):
- express@^5.1.0         - HTTP szerver keretrendszer
- mysql2@^3.15.3         - MySQL adatbázis driver
- cors@^2.8.5            - Cross-Origin Resource Sharing
- dotenv@^17.2.3         - Environment változók kezelése

Fejlesztési függőségek (DevDependencies):
- typescript@^5.9.3      - TypeScript fordító
- @types/node            - Node.js típus definíciók
- @types/express@^5.0.5  - Express típus definíciók
- @types/jest            - Jest tesztelő típus definíciók
- @types/cors@^2.8.19    - CORS típus definíciók
- tsx@^4.21.0            - TypeScript executor
- nodemon@^3.1.11        - Fájl figyelő (dev)
- tslint@^5.20.1         - Kód ellenőrzés

NPM SCRIPTS RENDELKEZÉSRE ÁLLNAK:
==================================

npm run build         - TypeScript fordítása JavaScript-re
npm run lint          - Kód stílusának ellenőrzése
npm start             - Server futtatása (build után)
npm run dev           - TypeScript futtatása közvetlenül (fejlesztés)
npm run watch         - TypeScript figyelő (otomata fordítás)


GYORS INDÍTÁS:
==============

1. Environment beállítása:
   cp .env.example .env
   (Szerkeszd az .env fájlt a saját adatbázis adataiddal)

2. Fejlesztésben futtatás:
   npm run dev
   
   vagy
   
   npm run build && npm start

BACKEND SZERKEZETE:
===================

lib/                    - Fordított JavaScript fájlok
src/
├── index.ts          - Server indítása
├── app.ts            - Express konfigurálása
├── db.ts             - MySQL pool
├── controllers/      - Kéréskezelés
├── services/         - Adatbázis logika
├── routes/           - API végpontok
├── models/           - Adatstruktúrák
├── dto/              - Adatátvitel
├── middleware/       - Auth, error handling
├── utils/            - Segéd funkciók
└── tests/            - Tesztek

node_modules/         - Installált csomagok


KÖVETKEZŐ LÉPÉSEK:
==================

1. .env fájl szerkesztése a saját adatbázis adataiddal
2. npm run dev - fejlesztéshez
3. localhost:3000 - server testéléséhez
4. Postman/Thunder Client - API teszteléshez


PORTOK & URLS:
==============

Server: http://localhost:3000
API Base: http://localhost:3000/api

API Végpontok:
- GET    /api/products
- GET    /api/categories
- GET    /api/users
- GET    /api/orders
- GET    /api/orderItems

(+ POST, PUT, DELETE mindegyik típusnál)


HIBAKEZELÉS:
============

Ha "Cannot find module" hibát kapsz:
→ npm install

Ha MySQL kapcsolat hiba:
→ Ellenőrizd a .env fájlt
→ Hogy a MySQL szerver fut-e

Ha TypeScript hiba:
→ npm run build
→ npm run dev
